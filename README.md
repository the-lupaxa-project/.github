<h1 align="center">The <em>.github</em> Repository</h1>

This repository is the central home for the shared standards, Community Health files, templates, documentation, and GitHub configuration used throughout
**The Lupaxa Project**.

GitHub provides special behaviour for repositories named .github. This repository uses those features to provide organisation-wide defaults and shared
resources that promote a consistent, secure, and maintainable experience across all **The Lupaxa Project** organisations and repositories.

## What You’ll Find Here

This repository contains:

- The public organisation profile
- Community Health files
- Organisation-wide documentation and standards
- Issue and pull request templates
- Shared GitHub configuration and supporting resources

## Organisation-Wide Documentation

The following documents define the standards that apply across **The Lupaxa Project**:

| Document                                     | Purpose                                                                                          |
| :------------------------------------------- | :----------------------------------------------------------------------------------------------- |
| [Code of Conduct](docs/GOVERNANCE.md)        | Expected behaviour and reporting procedures.                                                     |
| [How to Contribute](docs/CONTRIBUTING.md)    | Development workflow, coding standards, and contribution process.                                |
| [Governance](docs/GOVERNANCE.md)             | Project governance, roles, and decision-making.                                                  |
| [Maintainers](docs/MAINTAINERS.md)           | Maintainer responsibilities and expectations.                                                    |
| [Projects Overview](docs/PROJECTS.md)        | Overview of organisations, repositories, and project structure.                                  |
| [Security Policy](docs/SECURITY.md)          | Responsible disclosure and secure development practices.                                         |
| [Support Guide](docs/SUPPORT.md)             | How to obtain help and support.                                                                  |
| [Engineering Principles](docs/PRINCIPLES.md) | The core engineering philosophy and values that guide decision-making across The Lupaxa Project. |

These documents provide the common foundation for all **The Lupaxa Project** organisations and repositories. Individual repositories may extend or supplement
them where appropriate.

## Repository Purpose

This repository exists to provide a single, authoritative source for organisation-wide documentation and GitHub resources. Maintaining these shared assets in
one location helps ensure consistency across **The Lupaxa Project** while reducing duplication between repositories.
